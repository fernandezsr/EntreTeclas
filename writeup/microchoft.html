Introducción

Este write-up describe cómo abordamos el desafío de CTF X. En este desafío, tuvimos que explotar una vulnerabilidad en el servidor para obtener la bandera (flag).

Paso 1: Reconocimiento

El primer paso fue realizar un escaneo de puertos en el servidor para encontrar servicios expuestos, como vemos encontramos varios puertos abiertos, y vemos que es un windows 7 Home basic, y que el puerto 445 esta abiero. Por lo que podremos explotar la vulnerabilidad EternalBlue

El comando que hemos usado es el siguiente:

nmap -p- -sS -sC -sV --min-rate 5000 -n -vvv -Pn [IP] -oN all_ports

Paso 2: Explotación

En este caso usaremos metasploit para explotar la vulnerabilidad EternalBLue

Buscaremos Eternalblue en metasploit y configuraremos las opciones, en este caso el RHOSTS, la ip de la victima.

search Eternalblue
set RHOSTS [IP]

Paso 3: Escalada de privilegios

En este caso no nos hara falta escalar privilegios, ya que al explotar esta vulnerabilidad nos da privilegios elevados en el sistema
